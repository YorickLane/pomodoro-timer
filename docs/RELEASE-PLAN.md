## 发布检查清单

每次发布前确认：

- [ ] 版本号已更新 (`app.json`)
- [ ] 功能测试通过
- [ ] TypeScript 编译通过 (`npx tsc --noEmit`)
- [ ] 更新日志已编写
- [ ] Release Notes 已准备
- [ ] 商店描述是否需要更新
