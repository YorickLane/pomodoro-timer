# 🍅 番茄钟 - Pomodoro Timer

简洁、专注、高效。一款优雅的番茄工作法计时器，帮助你提升专注力和工作效率。

## ✨ 核心特性（计划）

- ⏱️ **经典番茄钟** - 25分钟工作 + 5分钟休息
- 📊 **今日统计** - 实时显示完成的番茄钟数量
- 🔔 **智能提醒** - 工作和休息时间的通知提醒
- ⚙️ **灵活设置** - 自定义工作和休息时长
- 🌙 **深色模式** - 护眼舒适的视觉体验
- 🔒 **隐私安全** - 数据仅本地存储
